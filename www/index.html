About:
這是我為了自己看 vtuber 需求做的小工具
在這個網頁可以查詢、追蹤想要的頻道，並且提供 iCal 連結讓其他行事曆 App 匯入
本意是模仿 Holodex，但是可以用來追蹤訂閱數通常低於 Holodex 註冊門檻的小頻道
如果想要追蹤訂閱數高於兩萬的 vtuber 頻道，還是推薦用功能比較完整的 Holodex

This is a basic tool made for myself.
It can track youtube channels ongoing/upcoming streams and provide a iCal resource to add to calendar apps.
This project is highly inspired by Holodex, but can be used to track small channels that doesn't meet the register threshold of Holodex.
If you want to track channels that has over 20k subs, Holodex is still recommended.
